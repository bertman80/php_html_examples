# google_recaptcha.php
A simple form that use Google reCAPTCHA

Google reCAPTCHA on a form.
Go to: https://www.google.com/recaptcha/admin
Make a new by clicking on +
Choose: reCAPTCHA v2
And then select an option like 'i'm nog a robot'
Add the domain name.
